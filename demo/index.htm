jQuery.bingMapify

a jquery library for bing maps v7

// code:
$('#one').bingMapify({
    mapPins: '/mapPins.json'
});
Use a json file to feed the pin points to the map 
// code:
$('#two').bingMapify({
    mapPins: [
        { lat: "38.000", lon: "-96.000", title: "my cool event", location: "some venue", address: "1234 Great St." },
        { Lat: "38.685509760012", Lon: "-96.50390625", Title: "another cool place", Location: "a different venue", Address: "5678 Amazing St." }
    ]
});
Set the mapPins inline in your script
you can capitalize the first character, or not. Just be consistent 
// code:
$('#three').bingMapify({
    mapPins: '/mapPins.json',
    infobox: {
        formatHtmlContent: '<div class="infoBox2"><h3>{title}</h3><hr>{location}<br>{address}<hr>{lat} | {lon}</div>'
        visibleOnLoad: true
    }
});
Customize the look of your infoBox
